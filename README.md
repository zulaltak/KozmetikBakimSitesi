# Kozmetik Bakim Sitesi
### Sadece HTML kodlari ile olusturulmus bir cilt bakimi/ kozmetik sitesi.
### Siteye dair her sey tarafimca hazirlanmistir.
### Problem teskil etmemesi icin urunlerin markalarini shop ile sildim.
### Iletisim kisminda yer alan konum urunlerinin fotografini kullandigim firmanin, mail adresi ise gercek degildir.
